Seletores CSS

No CSS nós possuímos os seletores por ID

Teste de seleção com o ID.


  • Seletor por tipo: Busca elementos por uma TAG HTML
  • Seletor por ID (#): Busca elementos através do atributo '#'
  • Seletor por Classe (.): Busca todos os itens com esse nome de classe
  • Seletor Universal (*): Seleciona todos os Elementos HTML em uma Pagina
  • Seletor Filho (>): Retorna os filhos diretos do 2º parâmetro
  • Seletor de atributo([atrib])
